# How to Apply the Top-Bottom/Average Conditional Formatting to Grid Cells


The **Top-Bottom** format conditions contained in the [FormatConditionTopBottom](https://docs.devexpress.com/Dashboard/DevExpress.DashboardCommon.FormatConditionTopBottom) class allow you to highlight a specific number of topmost/bottommost values.Â This example shows how to apply conditional formatting to the specified number of top/bottom values. The number of values can be specified as an absolute value or a percentage of all values.

The **Average** format condition contained in the [FormatConditionAverage](https://docs.devexpress.com/Dashboard/DevExpress.DashboardCommon.FormatConditionAverage) class allows you toÂ highlight values above or below an average value.
